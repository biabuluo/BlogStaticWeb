学习css笔记

什么是CSS

通俗来讲，HTML是对网页的界面进行结构布局的，但是光有布局还远远不够，还需要对布局进行美化，CSS就可以达到对网页界面美化的效果，让界面更加地美观

⭐CSS指层叠样式表（Cascading Style Sheets）

⭐样式定义如何显示HTML元素

⭐样式通常储存在样式表中

⭐把样式添加到HTML 4.0中，是为了解决内容与表现分离的问题

⭐外部样式表可以极大提高工作效率

⭐外部样式表通常储存在CSS文件中

⭐多个样式定义可层叠为一个

CSS背景

CSS 背景属性用于定义HTML元素的背景。

CSS属性定义背景的效果，以下为背景的常用属性值：

⭐backgound-color

⭐backgound-image

⭐backgound-repeat

⭐backgound-attachment

⭐backgoound-position

❗背景颜色: background-color 属性定义了元素的背景颜色. 页面的背景颜色使用在body的选择器中:


body{backgound-color:#b0c4de}

❗背景图像: background-image 属性描述了元素的背景图像. 默认情况下，背景图像进行平铺重复显示，以覆盖整个元素实体.

❗常用背景属性: 在实际的开发当中，我们常常要使背景图片居中对齐，并且铺满整个块元素。我们常做如下的设置：

.fakeimg {
background-image: url(..jpg);
background-size: cover;
background-position: center;
}

CSS创建

插入样式表的方法有三种:

⭐外部样式表(External style sheet)

⭐内部样式表(Internal style sheet)

⭐内联样式(Inline style).

CSS语法

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明:

⭐选择器通常是您需要改变样式的 HTML 元素。

⭐每条声明由一个属性和一个值组成。

⭐属性（property）是您希望设置的样式属性（style attribute）。每个属性有一个值。属性和值被冒号分开。

⭐CSS注释：

注释是用来解释你的代码，并且可以随意编辑它，浏览器会忽略它。

CSS注释以 "/*" 开始, 以 "*/" 结束

CSS文本

颜色属性被用来设置文字的颜色。

颜色是通过CSS最经常的指定：

⭐十六进制值 - 如: ＃FF0000

⭐一个RGB值 - 如: RGB(255,0,0)

⭐颜色的名称 - 如: red
